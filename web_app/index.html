How ICS prompt caching works
Without ICS — Naive
Select a scenario and click Compare
Every call sends all tokens again.
With ICS — Cached
Select a scenario and click Compare
Stable layers are cached once. Only dynamic layers are resent.
Formula:
Naive cost (N calls)  =  N × total_tokens
ICS cost    (N calls)  =  total_tokens + (N − 1) × dynamic_tokens
                            ↑ full first call         ↑ only dynamic on repeat calls
